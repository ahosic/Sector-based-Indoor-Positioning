getTurn

protected static fun getTurn(a: Point!, b: Point!, c: Point!): GrahamScan.Turn!

Returns the GrahamScan#Turn formed by traversing through the ordered points a, b and c. More specifically, the cross product C between the 3 points (vectors) is calculated:

(b.x-a.x * c.y-a.y) - (b.y-a.y * c.x-a.x)

and if C is less than 0, the turn is CLOCKWISE, if C is more than 0, the turn is COUNTER_CLOCKWISE, else the three points are COLLINEAR.

Parameters

a - Point!: the starting point.

b - Point!: the second point.

c - Point!: the end point.

Return
GrahamScan.Turn!: the GrahamScan#Turn formed by traversing through the ordered points a, b and c.
