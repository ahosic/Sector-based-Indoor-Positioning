getConvexHull

static fun getConvexHull(xs: IntArray!, ys: IntArray!): MutableList<Point!>!

Returns the convex hull of the points created from xs and ys. Note that the first and last point in the returned List<java.awt.Point> are the same point.

Parameters

xs - IntArray!: the x coordinates.

ys - IntArray!: the y coordinates.

Exceptions

IllegalArgumentException - if xs and ys don't have the same size, if all points are collinear or if there are less than 3 unique points present.

Return
MutableList<Point!>!: the convex hull of the points created from xs and ys.
